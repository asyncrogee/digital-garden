> [!info]
> Checks if `values` exists in an `array`
> - Returns __boolean__ `true` or `false`
> ```php
>$array = ['a', 'b', 'c', 'D', 'E', 'ab', 'bc', 'cd', 'b', 'd'];
> ```
> 
> ```php
> if (in_array('a', $array)) {
> 	echo 'Letter found';
> }
> ```
