#### Check if FILE EXISTS
>[!info] Check if `FILE` __EXISTS__:
>`file_exists('file name')`
>```php
>if (file_exists('foo.txt')) {
>	echo filesize('foo.txt'); // File Size
>} else {
>	echo 'File not found';
>}
>```
