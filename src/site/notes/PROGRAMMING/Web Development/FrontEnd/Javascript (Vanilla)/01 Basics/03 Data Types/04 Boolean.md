# 04 Boolean

---

Logical Type

Only has 2 values: **True** and **_False_**

```javascript
let nameFieldChecked = true; // yes, name filed is checked
let ageFieldChecked = false; // no, age field is not checked
```

```javascript
let isGreater = 4 > 1;

alert(isGreater); // true (the comparison result is "Yes")
```
